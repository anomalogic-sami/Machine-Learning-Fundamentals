# 🤖 Machine Learning Fundamentals Portfolio

Welcome! This repository is a collection of my core Machine Learning implementations. As a **Computer Engineering** student, I’m focused on the mathematical foundations of AI and how to eventually deploy these models onto **Embedded Systems** (Edge AI). 🚀

## 📂 Project Structure
Each folder contains the logic, the data, and the final report:

* **📉 Regression:** Linear and Logistic models to predict values and classify data.
* **🌳 Decision Trees:** Implementation of tree-based logic for complex classification.
* **⚡ Gradient Descent:** Understanding optimization and cost function minimization.
* **📊 Data & Reports:** Includes `.csv` datasets and `.docx` summaries of the results.

## 🛠️ Technical Stack
* **Language:** Python 3.x 🐍
* **Libraries:** NumPy, Pandas, Scikit-Learn, Matplotlib
* **Environment:** Jupyter Notebooks (.ipynb)
